# Player Reference Cards - Installation Instructions

## Image Files Required

You need to save the two player reference card images as PNG files in the `reference/` folder:

1. **player-reference-1.png** - The first reference card (showing REGENERATE, WARD, INVISIBLE, STRENGTHEN, BLESS, PUSH X, PULL X, PIERCE X, SHIELD X, RETALIATE X)

2. **player-reference-2.png** - The second reference card (showing WOUND, BRITTLE, BANE, POISON, IMMOBILIZE, DISARM, IMPAIR, STUN, MUDDLE, CURSE)

## How to Save the Images

### From the attached images:
1. Right-click on each image in the chat
2. Select "Save image as..."
3. Save them with the following names:
   - `player-reference-1.png` (first card)
   - `player-reference-2.png` (second card)
4. Place both files in: `c:\Clones\GloomHavenProject\GloomhavenScreenProject\reference\`

### Image Requirements:
- Format: PNG
- Recommended size: High resolution (at least 1200px wide) for clarity when displayed
- The images should have good contrast for readability

Detta projektet skapades ur ett behov av att snabba upp spelkvällarna i Gloomhaven.
Det tog tid att sätta upp brädet, hitta alla småkartongbitar för fällor etc etc, och spelgruppen ville inte att hela dungeonen skulle sättas upp direkt eftersom då blev det spoilers.

Resultatet blev ett digital hemsida som fungerar som ett spelbräde.
Små kartbitar som kan döljas med ett klick, terräng, fällor som kan tas bort via klick.
Markör för vart fiender ska stå när ett rum visas med färg för normal och elite version.
Och om den ska synas för spelgruppen baserad på 2,3 eller 4 spelare.

Textrutor för loot, basic info, special rules som kan klickas fram vid behov.
Just nu är denna klar, detta är en POC som har kodats med AI.

Den slutgiltiga versionen kommer vara en Serverbaserad Blazor App.
